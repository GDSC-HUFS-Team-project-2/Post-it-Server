# Post-it Project

GDSC 22-23 Team Project 2팀
<br/>친구 지인과 함께 즐길 수 있는 온라인 롤링페이퍼 서비스



## 프로젝트 개요
최근에 SNS에서 온라인 롤링페이퍼 서비스가 큰 인기를 얻고 있어 프로젝트 주제로 선정, SNS 링크 공유를 통해 친구와 지인끼리 쉽고 재미있게 '포스트잇'을 작성할 수 있는 온라인 롤링페이퍼 서비스를 개발해보고자 함.



* 개발기간: 2022.02.08 ~ 
* 개발인원: 총 5명
